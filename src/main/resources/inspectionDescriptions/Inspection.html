Detects user interface elements that increase cognitive load in Android applications.

The inspection analyzes Jetpack Compose code based on a set of cognitive load criteria (CLC1–CLC12). It identifies problems such as:

  • High information density (CLC1)
  • Low text readability (CLC2)
  • Poor visual accessibility (CLC3)
  • Small touch targets (CLC4)
  • Missing semantic attributes (CLC5)
  • Complex animations and transitions (CLC6)
  • Irrelevant or excessive information (CLC7)
  • Difficult error recovery (CLC8)
  • Deep UI hierarchy (CLC9)
  • Low component reuse (CLC10)
  • Complex navigation paths (CLC11)
  • Unstable recomposition behavior (CLC12)

These issues may negatively affect usability, increase user effort, and reduce overall efficiency of interaction with the application. 


// Example (high cognitive load):
Column {
    Text("Item 1")
    Text("Item 2")
    Text("Item 3")
    Text("Item 4")
    Text("Item 5")
}

This inspection is part of a cognitive load analysis system for Android user interfaces developed using Jetpack Compose.
